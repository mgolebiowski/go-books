# go-books

## Dependencies

- make
- go (min 1.22)
- nvm (https://github.com/nvm-sh/nvm)
- `nvm install` in the root dir to install node version (v20)

## How to run

- `make front_deps`
- `make front_build`
- `make run`

By default, the website is available under http://localhost:8080

## Run frontend tests
- `make front_test`
